API de Gerenciamento de Emails - Projeto Acadêmico 
Descrição:

Esta é uma API simples desenvolvida em c# com o objetivo de gerenciar emails de usuários.

A API permite:

Buscar emails cadastrados Adicionar novos emails Remover emails existentes

Tecnologias utilizadas:

.NET 8 Dotnet SDK 8 C# ASP.NET Core

Como executar o projeto:

Clone o repositório: git clone https://github.com/KauaRZ/NovaAPI.git Acesse a pasta do projeto: cd PrimeiraAPI Execute a aplicação: dotnet run A API estará disponível em: http://localhost:5229

Melhorias futuras: Implementar banco de dados Adicionar autenticação de usuários Criar validações de email Implementar atualização de dados (PUT) rodar API na nuvem
